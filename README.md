# gRPC-Test
gRPC server (with go) and client (with go, python)

## Start Server

```bash
# run server
go run server-script/main.go
```

## Run Client

```bash
# client by go
## run client
go run client-script/client.go

# client by python
## set PYTHONPATH
source python.env.sh
## run client
python client-script/client.py
```
